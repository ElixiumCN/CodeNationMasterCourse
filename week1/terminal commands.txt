Terminal commands

Cd - change directory
Ls - list directory contents
Mkdir - make directory
Touch - create new file in Git Bash Termianl
Rmdir - remove directory
Cd ~ go back to root folder
Cd .. Back one folder
Clear - clear terminal
Navigate previously used commands with arrow key
Run a file: Node xyz.js
npm start - start local web server
